# Assets de marque

- **`logo_tampon.png`** — logo principal (sceau à la fleur de lys), fourni depuis
  le Drive « Veille créative ». Intégré dans `components/wordmark.tsx` (header) et
  dans le hero de la page d'accueil, via `next/image`. En mode sombre il est
  inversé (`dark:invert`) pour rester lisible.

Le reste du moodboard (papiers, fleurs, couples…) sert d'inspiration : le rendu
papier/collage est recréé en CSS, sans rapatrier ces images dans le dépôt.

> Si un logo vectoriel (`.svg`) ou une version optimisée est fourni plus tard,
> le déposer ici et mettre à jour les deux `src` correspondants.
